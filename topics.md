# Разработка Интернет Приложений (РИП) 2025
## Темы вариантов

#### ИТ и Вычисления
1. RISC-V. `Услуги` - команды RISC-V ассемблера, `заявка` - программа для RISC-V с указанием значений в этой программе для каждой команды (какой регистр, какое условие и тд) с результатом
2. Команды ROS для управления роботом. `Услуги` - команды ROS, например библиотеки `rospy`, `заявка` - программа движения робота с конкретными значениями (градусы, расстояние) и вычисленным итоговым положением робота
3. Квантовый алгоритм в ввиде схемы. `Услуги` - однокубитные операции, `заявка` - квантовый алгоритм с указанием конкретных значений для гейтов и результат измерения кубита
4. Расчет времени рендера 3D-сцены на рендер-ферме. `Услуги` - конфигурации готовых серверов (CPU/GPU, количество ядер, объем RAM), `заявка` - расчет времени рендера для сцены с заданной сложностью (полигоны, текстуры, эффекты) на выбранных конфигурациях серверов.
5. Оценка стоимости облачного хостинга для веб-проекта. `Услуги` - тарифы облачных провайдеров (AWS EC2, Google Cloud, Azure, Яндекс, Сбер), `заявка` - расчет месячной стоимости выбранных вариантов исходя из прогнозируемой нагрузки: CPU, RAM, трафик, дисковое пространство.
6. Прогноз времени обучения нейронной сети. `Услуги` - выбранные пары архитектур моделей (ResNet, YOLO) и датасетов к ним (Cifar, ImageNet, coco), `заявка` - расчет времени обучения на графическом ускорителе для указанного количества эпох и размера батча.
7. Расчет пропускной способности беспроводной сети. `Услуги` - модели сетевого Wi-Fi оборудования, `заявка` - расчет максимальной скорости передачи данных с учетом помех, расстояния и протокола для выбранных моделей оборудования.
8. Оценка объема данных для архивирования. `Услуги` - алгоритмы сжатия данных, `заявка` - расчет времени и объема данных после сжатия для заданного объема исходных данных.

#### Химия и материалы
1. Производство аспирина. `Услуги` - этапы синтеза, `заявка` - расчет выхода чистого вещества с указанием описания и объема по каждому этапу
2. Синтез метана по реакции Сабатье. `Услуги` - исходные вещества и энергия, `заявка` - расчет массы метана по указанным количествам исходных веществ и энергии
3. Расчет давления в сосуде при нагреве идеального газа. `Услуги` - типы газов (Азот, Кислород, Гелий), `заявка` - расчет конечного давления при нагреве газа от начальной до конечной температуры в замкнутом объеме.
4. Расчет периода полураспада радиоактивного изотопа. `Услуги` - радиоактивные изотопы (Уран-235, Углерод-14, Йод-131 и др), `заявка` - расчет поверхностной активности (в беккерелях) оставшегося количества вещества от исходного через заданный промежуток времени.
5. Расчет текущих параметров малых модульных реакторов в конце периода. `Услуги` - модели малых модульных реакторов (РИТМ-200, NuScale (NPM) и тд), `заявка` - расчет электрической мощности, массы распавшегося топлива для выбранных реакторов за указанное время
6. Отклонение в пузырьковой камере для разных частиц. `Услуги` - исследуемые частицы (позитрон, альфа- и бета-частицы, и тд), `заявка` - расчет отклонения выбранных частиц с их энергиями для указанных параметров исследования (расстояние, параметры поля и тд)
7. Прогноз pH раствора после добавления щелочи. `Услуги` - типы кислот и щелочей (соляная кислота, гидроксид натрия и др), `заявка` - расчет конечного `pH` раствора после добавления заданного объема и концентрации реагентов.
8. Расчет фотоэффекта от падающего излучения. `Услуги` - диапазоны электромагнитного излучения (радио, ИК, видимый свет, УФ, рентген), `заявка` - расчет тока фотоэффекта от указанного распределения излучения по диапазонам волн.
9. Расчет длины де Бройля для частиц. `Услуги` - различные частицы (электрон, протон, нейтрино и тд), `заявка` - расчет длины волны/частоты по скорости частицы.
10. Расчет количества теплоты в реакции горения. `Услуги` - разные виды топлива (метан, пропан-бутан, ацетилен, водород), `заявка` - расчет количества теплоты (в кДж), выделившейся при полном сгорании заданного объема (при н.у.) выбранного топлива.
11. Прогноз растворимости осадка. `Услуги` - химические соединения (карбонат кальция, гидроксид меди(II), сульфат бария), `заявка` - расчет массы соединения, которое можно растворить в заданном объеме воды при указанной температуре.
12. Расчет концентрации ионов в растворе после смешивания. `Услуги` - растворы электролитов (соли, кислоты, щелочи) разной концентрации, `заявка` - расчет молярной концентрации ионов H⁺ или OH⁻ в растворе, полученном после смешивания заданных объемов выбранных растворов.
13. Расчет молярной массы газа. `Услуги` - различные газы с их молярными массами, `заявка` - расчет молярной массы неизвестного газа по данным о его массе и объеме при н.у. и указание концентрации для смеси выбранных двух газов.
14. Расчет массы исходного реагента для получения целевого продукта. `Услуги` - промышленные процессы (контактный метод получения SO₃, синтез аммиака, производство серной кислоты), `заявка` - расчет массы исходного сырья (например, пирита для получения SO₂), необходимой для производства заданной массы целевого продукта с учетом выхода.
15. Расчет объема газа, выделившегося в реакции карбонатов с кислотами. `Услуги` - различные кислоты, `заявка` - расчет объема (при н.у.) газа CO₂, который выделится при взаимодействии заданной массы карбоната кальция с избытком выбранной кислоты.
16. Определение массовой доли примесей в образце. `Услуги` - технические образцы веществ (известняк, мрамор, металлический цинк), `заявка` - расчет массовой доли примесей (%) в техническом образце, если при его обработке кислотой выделился заданный объем газа.
17. Расчет концентрации при титровании. `Услуги` - пары кислоты и щелочи для титрования (H₂SO₄, HCl, NaOH, KOH), `заявка` - расчет нормальной концентрации (н.) раствора кислоты, если на титрование заданного объема щелочи с известной концентрацией было израсходовано определенное количество мл кислоты.
18. Прогноз продуктов электролиза. `Услуги` - растворы солей для электролиза (NaCl, CuCl₂, Na₂SO₄), `заявка` - для выбранного процесса электролиза определить массу продуктов на катоде и аноде, пропуская через электролит заданный заряд аккумулятора (в мАч).
19. Расчет выхода продукта от теоретически возможного. `Услуги` - лабораторные синтезы (получение эфира, йодоформа, ацетилена), `заявка` - расчет практического выхода продукта в %, если из заданной массы исходного реагента было получено определенное количество продукта.
20. Определение формулы органического соединения по продуктам сгорания. `Услуги` - формулы простых органических соединений (алканы, алкены, спирты), `заявка` - по массе образца и объемам выделившихся CO₂ и H₂O рассчитать молекулярную формулу неизвестного органического вещества и указать молярную массу для перечисленных соединений.

#### Экономика
1. Рассчитать требуемые ресурсы для выращивания урожая в теплице на Севере. `Услуги` - виды ресурсов для теплицы с единицами измерения (площадь, электричество и тд), `заявка` - расчет по каждому показателю для указанного объема урожая
2. Расчет урожайности сада яблонь для указанных климатических условий в сезоне. `Услуги` - месяцы в году, `заявка` - расчет урожайности с указанными показателями в каждом месяце
3. Прогноз температуры на Земле в зависимости от содержания парниковых газов в атмосфере. `Услуги` - парниковые газы (углекисный газ, метан, оксид азота и тд), `заявка` - расчет средней температуры по указанной концентраци (в ppm) парниковых газов
4. Проценты по вкладам. `Услуги` - 12 месяцев с количеством дней, `заявка` - расчет процентов по вкладу, которые в каждый из указанных месяцев нужно выплатить
5. Расчет выработки электроэнергии солнечной фермой. `Услуги` - типы солнечных панелей (монокристаллические, поликристаллические, тонкопленочные), `заявка` - расчет месячной выработки энергии для указанного типа панелей, их количества/площади и инсоляции в регионе.
6. Прогноз мощности ветрогенератора на основе данных о ветре. `Услуги` - конкретные модели ветрогенераторов (разной мощности и высоты мачты), `заявка` - расчет выработки энергии по данным о средней скорости ветра за период для каждого варианта ветрогенераторов.
7. Расчет углеродного следа производства товара. `Услуги` - этапы жизненного цикла (добыча сырья, производство, логистика, утилизация), `заявка` - расчет общего объема выбросов CO₂-эквивалента для продукта с указанными параметрами на каждом этапе.
8. Оценка эффективности теплоизоляции здания. `Услуги` - типы изоляционных материалов (минвата, пенопласт, PIR-плиты), `заявка` - расчет теплопотерь и экономии энергии после утепления на указанной площади для каждого варианта материала.

#### Электронные устройства и электричество
1. Расчет объема трафика, потребляемого устройствами умного дома. `Услуги` - виды устройств, `заявка` - расчет на объем трафика для списка устройств с указанием количества устройств и их моделей для справки
2. Расчитать объем потребляемой датацентром электроэнергии в зависимости от оборудования в нем. `Услуги` - виды устройств (серверы, коммутаторы, СХД и тд), `заявка` - расчет потреблямой мощности с учетом охлаждения для датацентра со списком оборудования
3. Расчет потребляемой электроэнергии в квартире от используемых в доме устройств. `Услуги` - бытовые устройства с указанием мощности, `заявка` - расчет потреблямой мощности от списка устройств, температуры в этом месяце и количестве жителей
